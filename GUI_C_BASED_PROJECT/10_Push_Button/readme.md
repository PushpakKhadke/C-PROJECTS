# Push Button Project

This project is a basic Windows desktop application developed in C, featuring a window titled "PUSHPAK KHADKE" with a single push button. When clicked, the button triggers a message box displaying "Button clicked!".

## Technology Stack

`Win32 API` <br>
`C Programming` <br>
`Vs Code`

## How To Run This Project

                                
1. You will need to navigate to the location of your source code folder by using the "cd" command.

     ```shell
   cd source_code_location

2. Build exe file using following command.

    ```shell
   cl /Fe:app.exe Hello_Win.c

3. Run exe file using following command.

    ```shell
   app.exe
